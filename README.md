# Skill Clustering from Job Descriptions (Unsupervised Machine Learning)

This project performs **Unsupervised Clustering** on job descriptions to discover natural groups of skills in the job market.  
It is built using TF-IDF for text vectorization and K-Means for clustering.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib / Seaborn

##  Project Workflow
1. Data Loading & Inspection  
2. Text Cleaning  
3. Keyword Extraction  
4. TF-IDF Vectorization  
5. K-Means Clustering  
6. Cluster Interpretation  
7. Visualization + Results
